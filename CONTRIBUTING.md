# Guide de contribution Agri-Tech

## Objectif
Cette plateforme est conçue pour rassembler les contributions de plusieurs collaborateurs autour d’un même projet : Agri-Tech.

## Règles de contribution
1. Créer une branche dédiée à sa fonctionnalité.
2. Travailler sur un module ou une page cohérente avec le thème du projet.
3. Vérifier que les fichiers HTML/CSS/JS restent fonctionnels.
4. Ajouter la page ou le module à la navigation principale si nécessaire.
5. Ouvrir une Pull Request vers la branche principale.

## Structure attendue
- Les pages principales doivent rester accessibles depuis index.html.
- Le calculateur agricole doit rester central dans la plateforme.
- Les styles doivent conserver une identité visuelle cohérente.

## Workflow recommandé
- `git checkout -b feature/nom-de-la-fonctionnalite`
- `git add .`
- `git commit -m "Ajout de ..."`
- `git push origin feature/nom-de-la-fonctionnalite`

## Résultat attendu
Chaque contribution publiée sur GitHub apparaît comme une amélioration du projet collectif Agri-Tech.
