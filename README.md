# Agri-Tech - Plateforme collaborative

Cette plateforme regroupe les contributions de plusieurs collaborateurs autour d’un objectif commun : créer une solution Agri-Tech moderne, accessible et fonctionnelle.

## Modules disponibles
- Calculateur agricole
- Page d’accueil collective
- Pages de contribution des collaborateurs

## Structure du projet
- index.html : page principale de présentation du projet collectif
- Agri-Tech/calculator.html : module principal du calculateur agricole
- Agri-Tech/calculator.css : styles du calculateur
- Agri-Tech/calculator.js : logique du calculateur

## Intégration collaborative
Chaque contribution peut être ajoutée à cette plateforme en respectant la structure suivante :
1. Ajouter la nouvelle page dans le dossier principal du projet.
2. Relier la page depuis la page d’accueil collective.
3. Assurer la cohérence visuelle avec le thème Agri-Tech.
4. Vérifier que le calculateur reste central dans l’expérience utilisateur.

## Objectif
Unifier les efforts de l’équipe autour d’une seule plateforme, avec le calculateur agricole comme fonctionnalité principale et les autres modules comme compléments.
